Hubble

A GraphQL API for the Stellar Blockchain

Project Goals

  • No extra infrastructure overhead - Hubble runs alongside the official Horizon API and Database. No extra catchup or ingestionn is necessary!
  • Enhanced querying - Query transactions and operations for accounts between datetimes or ledgers
  • Familiar Schema - Query responses replicate the Horizon API where possible
